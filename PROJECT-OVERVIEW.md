# Project Overview

## 1. 프로젝트 개요

* **프로젝트명:** TunaChain (참치 초저온 콜드체인 모니터링 시스템)
* **핵심 기능:**
* 참치 이동 경로 및 실시간 온도($-60^\circ\text{C}$) 모니터링
* 임계치($-55^\circ\text{C}$) 이탈 시 실시간 경고 발생
* QR 코드를 통한 소비자용 이력 조회 페이지
* 데이터 위·변조 방지 검증 기능



---

## 2. 기술 스택 (웹 개발자 포트폴리오 맞춤)

| 영역 | 기술 스택 | 선택 이유 |
| --- | --- | --- |
| **Frontend** | React.js | 상태 관리 및 실시간 데이터 시각화(차트)에 유리 |
| **Backend** | Nest.js | 실시간 스트리밍 및 가벼운 시뮬레이터 서버 구축 적합 |
| **Database** | MongoDB + Redis | 이력 데이터 저장 및 실시간 경고 알림 캐싱 |
| **Blockchain** | 가상 네트워크 (Hardhat + Solidity) | *현실적 대안:* 가스비와 속도 문제를 고려해 로컬에서 Hardhat+Ethers.js로 스마트 계약을 모킹하여 구현 |
| **Real-time** | Socket.io 또는 WebSockets | IoT 센서 데이터의 실시간 대시보드 반영을 위함 |

---

## 3. 핵심 아키텍처 및 구현 범위

개인 프로젝트 환경에서 모든 데이터를 온체인(On-chain)에 올리면 비용과 성능 문제가 발생하므로, 대용량 데이터 분산 처리를 고려한 **하이브리드 방식**을 채택하여 설계합니다.

* **IoT 시뮬레이터 (Backend 스크립트):** 5초마다 참치의 현재 위치(가상 GPS)와 온도($-62^\circ\text{C} \sim -53^\circ\text{C}$ 사이 랜덤)를 생성하여 백엔드로 전송합니다.
* **백엔드 처리 (Hybrid 구조):**
* **Off-chain:** 전체 원본 데이터는 대용량 처리를 위해 일반 DB(MongoDB)에 저장합니다.
* **On-chain:** 유통 단계별 체크포인트(예: 어획 완료, 하역 완료, 가공 완료)의 핵심 요약본과 데이터 해시(Hash) 값만 블록체인 스마트 계약에 기록합니다.


* **프론트엔드 (대시보드):** 실시간 대시보드 차트 구현 및 온도 이탈 예외 상황 발생 시 UI 실시간 경고 알림을 연동합니다.

## 4. 단계별 마일스톤 (Step-by-Step Milestone)

### Step 1: 요구사항 정의 및 UI/UX 설계

* **화면 구조 설계:**
* **관리자용 대시보드:** 실시간 지도 위치, 온도 변화 실시간 그래프, 유통 단계 타임라인, 실시간 알림창 배치 정보 설계
* **소비자용 모바일 페이지:** QR 코드 스캔 시 진입하는 심플하고 직관적인 이력 검증 뷰 설계


* **데이터 모델링:** 참치 고유 ID, 선박/차량 정보, 타임스탬프, 실시간 온도, 위·경도(GPS) 구조 정의

### Step 2: 백엔드 아키텍처 및 IoT 시뮬레이터 구축

* **센서 데이터 시뮬레이터 개발:** Node.js의 `setInterval`을 활용하여 주기적으로 가상 환경 데이터를 생성하고 가공하는 백엔드 스크립트 작성
* **실시간 파이프라인 형성:** WebSockets(Socket.io)을 연동하여 시뮬레이터 데이터가 수집되는 즉시 프론트엔드로 브로드캐스팅하는 실시간 통신 환경 구축

### Step 3: 스마트 계약(Smart Contract) 개발 및 블록체인 연동

* **컨트랙트 로직 설계:** Solidity를 활용하여 "유통 단계 변경 시 핵심 데이터와 해시 값을 저장하고 검증"하는 스마트 계약 구현
* **Web3 연동 라이브러리 탑재:** 백엔드에서 특정 체크포인트 이벤트가 감지되었을 때 Ethers.js를 통해 로컬 블록체인 네트워크(Hardhat)로 트랜잭션을 발생시키는 시스템 연동

### Step 4: 프론트엔드 시각화 및 무결성 검증 기능 완성을 통한 통합

* **실시간 데이터 시각화:** Recharts 등 라이브러리를 활용하여 $-60^\circ\text{C}$ 기준선과 임계치($-55^\circ\text{C}$) 이탈 구간이 명확히 표시되는 실시간 실선 그래프 구현
* **무결성 검증 UI 구현:** 소비자가 QR 코드를 통해 웹 페이지에 진입했을 때, DB 데이터와 블록체인 온체인에 기록된 해시 값을 대조하여 "이 데이터는 변조되지 않았음"을 시각적으로 인증하는 로직 완성 및 최종 테스트
